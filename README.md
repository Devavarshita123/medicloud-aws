# MediCloud AWS

A production-ready cloud-native healthcare platform built to demonstrate real-world AWS Solutions Architect concepts through hands-on implementation.

## Technologies
- AWS
- Terraform
- Docker
- Amazon ECS
- React
- Spring Boot
- PostgreSQL
- GitHub Actions
- Amazon RDS
- Amazon S3
- AWS Lambda
- Amazon SQS
- AWS Step Functions
- Amazon CloudWatch
- AWS X-Ray
- AWS CloudTrail
- Amazon Route 53
- Amazon CloudFront
- AWS WAF

## Goal
To build an enterprise-grade healthcare platform following AWS Well-Architected Framework principles while gaining practical experience in cloud architecture, scalability, security, monitoring, automation, and DevOps.
